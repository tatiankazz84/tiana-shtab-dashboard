# Штаб задач Татьяны

Mobile-first HTML-дашборд по логике урока ЛАБС: Google Sheets как база, HTML как визуальная витрина, дальше публикация через GitHub Pages и подключение к Telegram mini app.

## Текущая версия

Файл дашборда: `index.html`.

Структура первого экрана:

- аватарка Тианы;
- короткие KPI;
- Сегодня;
- В работе;
- Обзор с графиками;
- Горит;
- Решить;
- нижняя навигация под телефон.

## Важный файл изображения

Для корректного отображения аватарки в опубликованной версии нужно положить картинку сюда:

`assets/tiana-avatar.jpg`

Именно это имя уже прописано в HTML.

## Публикация через GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить содержимое этой папки в репозиторий.
3. Проверить, что есть:
   - `index.html`;
   - `.nojekyll`;
   - `assets/tiana-avatar.jpg`.
4. Включить GitHub Pages: Settings → Pages → Deploy from branch → main → root.
5. Открыть выданную ссылку и проверить с телефона.

## Следующий слой

После публикации:

- подключить кнопку/ссылку в Telegram;
- позже вынести данные в отдельный JSON или подтягивать из Google Sheets;
- не публиковать приватные финансы и клиентские данные без закрытого доступа.
