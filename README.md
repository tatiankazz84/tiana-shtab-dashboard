# Штаб задач Татьяны — визуальная витрина

## Роль

- **Источник истины:** Google-таблица `00_Штаб задач`.
- **Эта папка:** код и материалы Mini App.
- **Публичная страница:** личная визуальная витрина, не место хранения финансов, клиентских данных или личных заметок.

## Синхронизация

Данные из Google-таблицы регулярно попадают в опубликованный интерфейс. Mini App работает только для Татьяны: он показывает три вещи — срочное на сегодня, решения, которые нужны лично ей, и выбранные планы. Внутренние задачи Тианы, финансовые детали и клиентские данные в витрину не попадают. Статус задачи меняется в `00_Штаб задач`: Татьяна сообщает о выполненном, Тиана обновляет один источник истины.

Двустороннюю синхронизацию можно строить только после отдельного решения о закрытой авторизации и защите данных.

## Внутри проекта

- `index.html` — опубликованная витрина;
- `scripts/refresh_dashboard.py` — обновление данных из штаба;
- `assets/` — нужные изображения;
- `Architecture.md`, `scripts/` и `requirements-dev.txt` — технические материалы для поддержки витрины.

Публикация или изменение внешнего интерфейса — только по явно согласованному сценарию. Регулярное обновление текущей уже согласованной витрины разрешено отдельным расписанием.
